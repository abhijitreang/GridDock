Capabilities

Everything you need for virtual screening

Batch Ligand Processing

Import SMILES files and automatically generate 3D conformers, add hydrogens, and minimize energy using MMFF94 or UFF force fields.

Smart Grid Box

Specify active site residues and GridDock automatically computes the optimal docking grid box center and dimensions around the binding pocket.

AutoDock Vina Engine

Powered by AutoDock Vina for fast and accurate molecular docking. Extracts binding affinities and exports results to a summary CSV.

Real-Time Monitoring

Track docking progress with a live process log, progress percentage, and the ability to stop and restart at any time.

Workflow

From SMILES to binding affinity in 4 steps

1

Load inputs & settings

Select your SMILES file containing ligand strings, choose an output folder, and pick an energy minimization force field (MMFF94 or UFF).

2

Configure docking

Browse for your receptor PDBQT file, locate the Vina executable, and enter active-site residue numbers (e.g. 10, 20, 30) to define the binding pocket.

3

Run the experiment

Hit START DOCKING and monitor the real-time process log. GridDock handles ligand conversion, grid computation, and Vina execution sequentially.

4

Analyze results

Open your output folder to find docked PDBQT files and a Docking_Summary.csv with the best binding affinity (kcal/mol) for each ligand.

Walkthrough

How to use GridDock

A visual guide through the interface, from launch to results.

1

Load inputs and settings

Browse to your SMILES file, choose an output folder, and select the energy minimization force field (MMFF94 or UFF).

2

Configure the receptor

Load your receptor .pdbqt file, point to the Vina executable, and enter the active-site residue numbers that define the binding pocket.

3

Run docking

Click START DOCKING. The Process Log streams ligand conversion, grid computation, and Vina runs in real time. Stop and restart any time.

4

Review the results

Open the output folder to find docked PDBQT files and Docking_Summary.csv with the best binding affinity (kcal/mol) per ligand.
