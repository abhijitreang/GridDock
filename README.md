GridDock: An Automated GUI-Based Tool for High-Throughput Virtual Screening

Developed by
Department of Molecular Biology & Bioinformatics
Tripura University

To effectively use this tool, ensure these files are ready: 
1. SMILES File: Text file with ligand SMILES strings. 
2. Receptor File: Target protein structure in .pdbqt format. 
3. AutoDock Vina: The vina executable. 
4. Active Site Info: Specific residue numbers for binding. 
How to Use GridDock 
1. Input & Settings 
1. SMILES File: Browse to select your ligand file. 
2. Output Folder: Browse to select or create the folder for GridDock results. 
3. Minimization: Select the force field (e.g., MMFF94) from the dropdown for energy 
minimization, which prepares the molecules for docking. 
2. Docking Setup 
1. Receptor (.pdbqt): Load your target protein file via Browse. 
2. Vina Executable: Locate and load the vina.exe (or similar) docking engine file via 
Browse. 
3. Active Site Residues: Enter binding pocket residue numbers, separated by commas 
(e.g., 10, 20, 30). 
3. Run the Experiment 
1. Start: Click the green START DOCKING button. 
2. Monitor: Watch the Process Log on the right for real-time computer activity. 
3. Stop: Click the red STOP button to cancel or correct mistakes. 
4. Results 
1. Once the docking process is complete, open the Output Folder you selected.
