About GridDock

GridDock is an automated graphical interface for high-throughput molecular docking and ligand preparation. Built with Python and Tkinter, it wraps the full docking pipeline, from SMILES input to binding affinity output, into a single click-and-run workflow.

Built With

  • Python 3 & Tkinter GUI
  • RDKit for cheminformatics
  • Open Babel for format conversion
  • AutoDock Vina docking engine
  • Multithreading for non-blocking UI

Input Requirements

  • SMILES file (.txt) with ligand strings
  • Receptor structure in .pdbqt format
  • AutoDock Vina executable
  • Active-site residue numbers

Outputs

  • Docked ligand PDBQT files
  • Docking_Summary.csv with affinities
  • Binding affinity in kcal/mol
  • Per-ligand log files

Under the Hood

Ligand Processing Pipeline

1

Fragment Cleaning

Multi-fragment SMILES are split and only the primary fragment is retained.

2

Molecule Construction

RDKit builds the molecule, sanitizes it, and adds hydrogens explicitly.

3

3D Conformer

ETKDGv3 algorithm generates a low-energy 3D conformer for each ligand.

4

Energy Minimization

User-selectable force field (MMFF94 or UFF) optimizes the geometry.

5

PDB → PDBQT

Open Babel converts the ligand using Gasteiger charges for Vina compatibility.

6

Docking & Scoring

Vina runs the dock. Best binding affinity is extracted and logged to CSV.
